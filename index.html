LitXR: Render LitElement Web Components in 3D

LitXR is a set of two types of components. First, a LitElement extended base class (LitXR) to help render components in 3D. Second, a "surface" component to consume a tree of LitXR components. This project also uses <babylon-scene> to provide a component to hold a 3D canvas, the Babylon.js 3D library, and an easy way to set up the 3D scene. LitXR also provides helpers to glue it all together
